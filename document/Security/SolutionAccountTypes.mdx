# Solution Account Types
The RODE platform utilises user specific role based access controls to provide the security model for the platform.

There are three types of end user accounts present within the RODE security design. They are:

| Account Type | Description |
| :--- | :--- |
| System Administrator | The RODE system administration. System administrators are responsible for the management of the system information and metadata. System administartors have unrestricted access to the RODE user interface. |
| Administrator | The organisational administrator for a RODE partner. This user can create / alter affilations for end users related to their organisation. They are also able to access the entire user interface for their organisation. |
| End User | The RODE end user is any person accessing the RODE user interface not predefined as an administrator. The RODE end user has all user interface paths and data requests restricted according to the MINData governance rules |
